Program: CallGhost
Section 1: Explanation on the program and what id does
Main idea: A program which listens an interface and provides SRT and TXT output from it, for later passing to other tools.
Working instructions
When program is started will try to locate all resources needed for this task downloading or requesting the path to required tools such as ffmpeg for audio conversion if needed
It will present a menu with all possible input devices to capture sound from, it will display which ones are emiting sound or not with a small indicator of being silent or active
It will capture audio from the selected device on previous menu
When audio goes silent for a certain ammount of seconds, it beep three times to default output device and will show the message paused to allow user to continue recording once audio is restablished or end the program
Program ending, can be done anytime pressing esc in menu, it will stop and generate ogg, srt and txt files in the desired output folder
Program will read init parameters from a ini file, but will allow to override by passing them as parameter, create a ini file  with comments and defaults
You can read file transcribe_summary.py to check which are the parameters, group in sections
init config file notes and sections
# app config and paths to tools
# default capture mode or live device with their parameters
# default capture spects format (16khz mono)
# default capture outputs recording/YYY_MM_DD_recording.ogg, recording/YYY_MM_DD_recording.srt, recording/YYY_MM_DD_recording.txt
